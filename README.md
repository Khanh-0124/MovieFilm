## Run Locally

Clone the project

```bash
  https://github.com/Khanh-0124/MovieFilm.git
```

Go to the project directory

`cd MovieApp`

Install dependencies using `npm install`  or `yarn install`

Start the server

```bash
  npm run start
```

Run android use command `npx react-native run-android` or `yarn android`
## Tech Stack

**Client:** React Native

**Server:** FireBase
