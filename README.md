## Run Locally

Clone the project

```bash
  https://github.com/Khanh-0124/MovieFilm.git
```

Go to the project directory

```bash
  cd MovieApp
```

Install dependencies

```bash
  npm install
```
or
```bash
  yarn install
```
Start the server

```bash
  npm run start
```

Run android


```bash
  npx react-native run-android
```
or
```bash
  yarn android
```
